== Description ==

**Optimizes** your Wordpress blog for Search Engines

Features:

    * Automatically optimizes your titles for search engines.
    * Generates META tags automatically
    * You can override any title and set any META description and any META keywords you want.
    * Add index, noindex, follow or nofollow, noodp, noydir, noarchive, nosnippet meta tags to any post/page.
    * Custom field for Google Webmasters tools site verification meta tag.
    * Basically it means you can stop putting duplicate content on your site and provide the search engines with what it believes to be unique content.
    * this plugin detects all ways to access your original WordPress feeds and redirects them to your FeedBurner feed so you can track every possible subscriber.
    * Manual Ping-O-Matic
    * FB Like Button
    * Share Button
    * Remove WP versions
    * Eliminating Error Messages
    * Speed-up page updating, when using custom permalinks and a lot of pages.

Information / support Visit: **[http://www.papadestra.com/](http://www.papadestra.com/)**

== Installation ==

1. Upload Directory `bye_maridjan` to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. A new menu will appear! Bye Maridjan
